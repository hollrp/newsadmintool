Требуется написать страницу администратора, на которой можно добавить новость, закачать картинку к новости, пометить новость как не актуальную, удалить, отредактировать.
Базовые требования:
-Браузерное приложение под стационарный компьютер с обычной версткой.
-При загрузке появляется список добавленных новостей по группам в порядке убывания.
-Можно отредактировать новость.
-Можно удалить новость.
-Можно сделать новость неактуальной, тем самым не показывать в приложении.
-Реализован поиск новости по заголовку (head)
-Данные отображаются на пользовательском интерфейсе с помощью технологии JQGrid
-Поддержка ТОЛЬКО Русского языка интерфейса.
-Валидация полей на стороне клиента. Дата валидируется с помощью regex.
- Используемая база данных –MYSQL
-Изображения хранятся в базе данных в байткоде.
DAO слой написан на nodejs.
Структура таблицы в БД:
	Название: 
News- название таблицы в БД
Поля:
	Id –уникальный идентификатор новости
	head – Заголовок новости. VARCHAR(200)
	body- Тело новости. TEXT
	picture –TEXT
	use_flag –TINY_INT(1)
	news_date DATE
